# YOLO-Mouse-Minipulation-CSHARP-Edition
YoloMouse — a Windows tool that detects an on-screen object with a YOLOv10 ONNX model and smoothly moves the cursor onto it, via Windows mouse events or a real RP2040/RP2350 USB-HID mouse. GUI with confidence, target, activation, and smoothing controls. Built in C++, Python, Rust, and C#.
